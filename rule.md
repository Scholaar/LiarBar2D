# 骗子酒馆游戏规则

## 一、游戏目标

成为最后存活的玩家。 玩家需要通过：

- 撒谎
- 伪装
- 质疑
- 推理 让其他玩家在俄罗斯轮盘中被淘汰。

------

# 二、游戏人数

4人

------

# 三、游戏道具

## 扑克牌

游戏使用以下牌型：

- A
- K
- Q
- J
- **Joker（小丑牌）**

Joker总数2张，其余总数各6张

## 系统随机发牌。

## 左轮手枪

每位玩家拥有：

- 1把左轮手枪
- 6个弹槽 失败玩家需要进行俄罗斯轮盘。

------

# 四、游戏流程

------

## 1. 准备阶段

玩家进入房间后点击：

```text
准备
```

房主点击：

```text
开始游戏
```

## 游戏正式开始。

## 2. 发牌阶段

系统为每位玩家发放：

```text
5张手牌
```

## 玩家只能看到自己的牌。

## 3. 指定目标牌

每轮开始时： 系统随机指定一种目标牌。 例如：

```text
本轮目标牌：Q
```

## 本轮中： 玩家理论上应该出 Q。 但允许撒谎。

## 4. 出牌阶段

玩家按顺序行动。 当前玩家需要：

### 选择出牌数量

可选择：

- 1张
- 2张
- 3张

------

### 声明所出的牌

例如：

```text
"我出了2张Q"
```

------

## 注意

玩家真实出的牌： 其他玩家无法看到。 因此： 玩家可以：

- 真出目标牌
- 部分撒谎
- 完全撒谎 例如： 声明：

```text
2张Q
```

实际：

```text
A + K
```

------

# 五、Joker（小丑牌）规则

------

## Joker 的特殊能力

Joker **只能充当目标牌**使用。

出牌时声明为当前轮的目标牌，视为真话，不算撒谎。

例如：本轮目标牌为 Q，手持 Joker，可声明为 Q 出牌，若被质疑则质疑者失败。

------

## Joker 的质疑规则

若被质疑者出牌中包含 Joker：

- Joker 充当目标牌声明，视为**未撒谎**
- 质疑者失败，进入俄罗斯轮盘

------

## Joker 的数量

每副牌包含：

```text
2张 Joker
```

------

## Joker 的限制

- Joker **不能**作为目标牌被系统指定（目标牌仅从 A、K、Q、J 中随机选取）。
- Joker **只能**声明为目标牌出牌，不可用于其他声明。

------

# 六、质疑机制

## 下一位玩家可以选择：

## 1. 相信（Pass）

## 游戏继续。 轮到下一位玩家行动。

## 2. 质疑（Challenge）

## 系统公开上一位玩家实际出的牌。 随后进行判定。

# 七、质疑判定

------

## 情况1：上一位玩家撒谎

例如： 声明：

```text
2张Q
```

实际：

```text
A + K
```

则：

- 被质疑者失败
- 进入俄罗斯轮盘

------

## 情况2：上一位玩家没有撒谎

例如： 声明：

```text
2张Q
```

实际：

```text
Q + Q
```

则：

- 质疑者失败
- 进入俄罗斯轮盘

------

# 八、俄罗斯轮盘

失败玩家需要：

```text
开枪一次
```

## 系统随机判定是否中枪。

## 初始规则

左轮：

- 6个弹槽
- 初始1颗子弹 即：

```text
中枪概率 = 1/6
```

------

# 九、危险成长机制

## 每次轮盘幸存后： 下次开枪危险增加。

## 示例

| 开枪次数 | 子弹数量 | 中枪概率 |
| -------- | -------- | -------- |
| 第1次    | 1        | 1/6      |
| 第2次    | 2        | 2/6      |
| 第3次    | 3        | 3/6      |
| 第4次    | 4        | 4/6      |

------

# 十、淘汰规则

若玩家中枪： 则：

- 玩家立即淘汰
- 无法继续行动
- 进入观战状态

------

# 十一、回合结束

当：

- 所有玩家行动结束 或者：
- 本轮提前结束 系统将：
- 清空桌面牌
- 重新发牌
- 开启下一轮

------

# 十二、胜利条件

当场上仅剩：

```text
1名玩家
```

## 该玩家获胜。

# 十三、超时规则

每位玩家拥有：

```text
15秒行动时间
```

若超时： 系统将：

- 自动随机出牌

------

# 十四、断线规则

玩家断线后：

- 游戏继续
- 系统保留玩家状态 若玩家在规定时间内重连：
- 自动恢复房间
- 自动同步游戏状态 超时未重连：
- 直接判负

------

# 十五、游戏结束

当仅剩最后一名存活玩家时： 游戏结束。 系统显示：

- 胜利玩家
- 淘汰顺序
- 本局统计
